# Roles and Responsibilities

## 1) Terrain / DTM Engineer
Owns:
- input QA
- Stage 1 prepare
- DTM stage

Deliverables:
- preflight and prepare summaries
- DTM artifacts

## 2) Hydrology Engineer
Owns:
- hydrology baseline stage
- quality checks for hydrology outputs

Deliverables:
- hydrology intermediate outputs

## 3) ML Engineer
Owns:
- feature engineering stage
- model training
- metric reporting

Deliverables:
- metrics.json
- predictions.csv
- feature_importance.csv

## 4) Optimization Engineer
Owns:
- drainage routing optimization stage
- output consistency checks

Deliverables:
- proposed_drainage_lines.geojson
- optimization summary and design parameters

## 5) Systems / Documentation Engineer
Owns:
- environment consistency (`.venv` policy)
- stage execution sequencing
- logs and final documentation quality

Deliverables:
- updated docs
- activity log
- final readiness checklist
